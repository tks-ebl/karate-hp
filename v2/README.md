# 矢頭派糸東流修交会 修義館 サイト

GitHub Pages ですぐ公開できる、修義館全体から各ページへ案内する空手教室向けのサイトです。

## ファイル構成

- `about.html`: 道場説明
- `practice.html`: 稽古紹介
- `locations.html`: 稽古場所一覧
- `fees.html`: 月謝・入会案内
- `faq.html`: Q&A
- `styles.css`: デザインとレスポンシブ対応

## 公開方法

1. GitHub に push します。
2. リポジトリの `Settings` → `Pages` を開きます。
3. `Deploy from a branch` を選び、公開ブランチを `main` / `/ (root)` に設定します。
4. 数分待つと公開 URL が発行されます。

## 先に差し替える項目

- 各道場名
- 各道場の稽古場所、曜日、時間
- 月謝、入会金
- 電話番号
- 各道場の SNS / 連絡先 URL
